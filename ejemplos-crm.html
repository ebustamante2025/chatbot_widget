🤖 Guía de Integración CRM - Chatbot Isa

Esta página contiene ejemplos prácticos para integrar el widget del chatbot Isa en diferentes sistemas CRM. Elige el método que mejor se adapte a tu plataforma.

Método 1: Iframe Directo

✅ Recomendado para la mayoría de CRMs

El método más simple y compatible. Solo copia y pega este código en tu CRM:

<!-- Widget Chatbot Isa --> <iframe id="isa-chatbot-widget" src="https://tudominio.com/widget-chatbot/index.html" style=" position: fixed; bottom: 0; right: 0; width: 400px; height: 650px; border: none; z-index: 9999; box-shadow: 0 -4px 20px rgba(0, 0, 0, 0.2); border-radius: 20px 20px 0 0; " title="Chatbot Isa" allow="microphone" frameborder="0" ></iframe> <style> @media (max-width: 768px) { #isa-chatbot-widget { width: 100%; height: 100vh; border-radius: 0; } } </style>
Nota: Reemplaza https://tudominio.com/widget-chatbot/index.html con la URL real donde hayas desplegado el widget.

Método 2: Script de Carga Dinámica

✅ Ideal para integraciones más flexibles

Usa el script loader para una integración más controlada:

<!-- Configuración opcional --> <script> window.IsaWidgetConfig = { widgetUrl: 'https://tudominio.com/widget-chatbot/index.html', position: 'bottom-right', // 'bottom-right' | 'bottom-left' zIndex: 9999, width: 400, height: 650 }; </script> <!-- Cargar el widget --> <script src="https://tudominio.com/widget-chatbot/isa-widget-loader.js"></script>

Ejemplos por Plataforma CRM

📊 Salesforce

En un componente Lightning:

<aura:component> <iframe src="https://tudominio.com/widget-chatbot/index.html" style="position: fixed; bottom: 0; right: 0; width: 400px; height: 650px; border: none; z-index: 9999;" title="Chatbot Isa" /> </aura:component>

💼 Microsoft Dynamics 365

Como Web Resource o control de iFrame en formularios.

🔵 HubSpot

En el editor de plantillas, agrega el iframe antes de </body>.

📝 WordPress (con plugins CRM)

Usa el plugin "Insert Headers and Footers" o edita footer.php.

⚙️ Configuración Avanzada

Cambiar Posición

// Cambiar a esquina inferior izquierda var iframe = document.getElementById('isa-chatbot-widget'); iframe.style.right = 'auto'; iframe.style.left = '0';

Cambiar Tamaño

// Personalizar dimensiones var iframe = document.getElementById('isa-chatbot-widget'); iframe.style.width = '350px'; iframe.style.height = '600px';

Controlar Visibilidad

// Ocultar el widget IsaWidget.hide(); // Mostrar el widget IsaWidget.show(); // Eliminar el widget IsaWidget.remove();

🚀 Pasos para Desplegar

  1. Construir el widget: npm run build
  2. Subir archivos: Sube la carpeta dist/ a tu servidor
  3. Configurar URL: Asegúrate de que la URL sea accesible públicamente
  4. Integrar en CRM: Usa uno de los métodos mostrados arriba
  5. Probar: Verifica que el widget se muestre correctamente
💡 Tip: Para producción, considera usar un CDN para mejorar el rendimiento y la disponibilidad del widget. También asegúrate de configurar HTTPS para mayor seguridad.
